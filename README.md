# ** Дипломный проект по теме unit-тесты на курсе "Автоматизатор тестирования на Java"**

#### **Стэк проекта**

Java11, JUnit4, Maven, Jacoco, Mockito

#### **Как запустить**

``` shell 
mvn clean test
```
``` shell 
mvn verify
```
## **Техническое задание**

### **Покрыть тестами классы:**
1. Bun
2. Burger
3. Ingredient
4. IngredientType

**Процент покрытия должен составлять:**

Не менее 70%
